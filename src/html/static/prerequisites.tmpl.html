Development: prerequisites

This application is created using a simple http-server as back-end. In real world situations you should use some more powerful and flexible server like NodeJS (JavaScript), Play! (Scala and Java), Spray (Scala API server), etc. Front-end is decoupled from the back-end and based on HTML, JavaScript and CSS. Communication between the back-end and the front-end is done through the REST API.

Front-end

Front-end is based on HTML, JavaScript and CSS. There is no server side rendering. Whole front-end is deployed to the browser as static pages and uses REST JSON to obtain data from the server.

To develop the front-end, following needs to be downloaded and installed:

  • Node.js
Once Node.js is installed, please run the following to download the rest of programs and dependencies. 

npm install -g grunt-cli
npm install -g gulp
npm install -g json-server
npm install

Front-end tasks (dependencies, uglification, concatenation, testing, etc) are done using Node.js which, in turn, runs Bower, Grunt and Gulp.

Additional programs are installed using Node.js. Bower is used to download dependencies. Testing is performed with Grunt which, in turn, executes Jasmine specs. The rest of tasks is done with Gulp.
