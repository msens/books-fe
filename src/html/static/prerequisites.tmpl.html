Development: prerequisites

This sample is based on AngularJS as client side single page application MVC (or MVW) and Bootstrap CSS for style definition. The application is decoupled from the server thus allowing us to use any server side technology or framework. The only relation with the server is through JSON API and static files (HTML, CSS, JS). There is no server-side rendering.

This application is created using a simple http-server as back-end. In real world situations you should use some more powerful and flexible server like NodeJS (JavaScript), Play! (Scala and Java), Spray (Scala API server), etc.

This sample is used only for the development of the front-end part of the application. To facilitate easier and server-side independent development, http-server and json-server NodeJS modules are used.

Front-end

To develop the front-end, following needs to be downloaded and installed:

  • Node.js
Once Node.js is installed, please run the following to download the rest of programs and dependencies. 

npm install -g grunt-cli
npm install -g gulp
npm install -g json-server
npm install

Front-end tasks (dependencies, uglification, concatenation, testing, etc) are done using Node.js which, in turn, runs Bower, Grunt and Gulp.

Additional programs are installed using Node.js. Bower is used to download dependencies. Testing is performed with Grunt which, in turn, executes Jasmine specs. The rest of tasks is done with Gulp. Static files are server using NPM module http-server. JSON API mocks are created with NPM module json-server.
